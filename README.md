## 🎮 О программе

**Taskbar Hero Notifier** — это незаменимый помощник для игроков в [Taskbar Hero](https://store.steampowered.com/app/2813560/Taskbar_Hero/). Программа работает в фоновом режиме, читает локальные сохранения игры и предоставляет удобный интерфейс для отслеживания вашего прогресса.

### ✨ Главные возможности:
- **Уведомления в Telegram**: Бот сам напишет вам, когда ваши сундуки готовы к открытию. Больше никаких пропущенных наград!
- **Живой Дашборд**: Красивый веб-интерфейс, где можно посмотреть свою статистику, прогресс прохождения карт, инвентарь и надетые вещи в реальном времени.
- **Поддержка нескольких окон**: Привязывайте разные аккаунты к одному Telegram-боту и получайте уведомления со всех окон игры.
- **Автообновления**: Программа сама найдет новую версию и уведомит вас о доступном обновлении прямо в Windows.

## 🚀 Как начать пользоваться?

1. Перейдите в раздел **[Releases](https://github.com/Niobiumru/tbh-releases/releases/latest)** справа.
2. Скачайте последнюю версию файла `tbh_notifier_vX.X.X.exe`.
3. Запустите скачанный файл (убедитесь, что сама игра Taskbar Hero уже запускалась на этом ПК).
4. Программа откроет удобный Дашборд прямо в вашем браузере!

> **Примечание:** Этот репозиторий используется **исключительно** для хранения готовых сборок (`.exe`). Если вы хотите посмотреть исходный код проекта, добро пожаловать в главный репозиторий: **[Niobiumru/tbh](https://github.com/Niobiumru/tbh)**.
